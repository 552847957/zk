Use the Theme that Depends on Locale

It is common to use the larger font only for Chinese locales. You can configure WEB-INF/zk.xml to use it as follows.

<desktop-config><disable-theme-uri>~./zul/css/norm*.css.dsp*</disable-theme-uri><theme-uri>~./zul/css/norm**.css.dsp</theme-uri></desktop-config>
