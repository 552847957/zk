Events

All XHTML components support the following events, but whether it is applicable still depends on the browsers. For example, onChange is meaningless to non-input components, say body and div. You have to consult the HTML standard[53]

Event Name

Components

Description

onChange

all

Event: org.zkoss.zk.ui.event.InputEvent

Denotes the content of an input component has been modified by the user.

onClick

all

Event: org.zkoss.zk.ui.event.MouseEvent

Denotes user has clicked the component.


[53] http/www.w3c.org
