ZK Is Presentation-Tier Only

ZK is aimed to be as thin as the presentation tier. In addition, with the server-centric approach, it executes all codes at the server, so connecting database is no different from any desktop applications. In other words, ZK doesn't change the way you access the database, no matter you use JDBC or other persistence framework, such as Hibernate[62].
