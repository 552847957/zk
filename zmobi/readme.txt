zmobi - ZK Mobile Client
----------------


* ZK Attributes for Application Descriptor

ZK-Server-URL:
	If not specified, a prompt is shown to let user enter an URL, or select
	from a list of URLs that he ever entered.

	If specified, it is so-called Home Page. In other words, zmobi directly browses
	to the specified website when started.

	No matter specified or not, user can change the URL dynamically later by
