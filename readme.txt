README: Build ZK


-----
zkmob

	ZK Mobile Client is a MIDlet depending on MIDP.

	1.	Download Sun Java Wireless Toolkit (WTK) at
		http://java.sun.com/products/sjwtoolkit/download-2_5.html

	2.	Make a symbolic link to the directory you installed WTK (in Cygwin)
		cd /usr
		ln -s /cygdrive/c/WTK2.5 WTK

	To test with WTK, you have to execute
		bin/deploy zkmob
