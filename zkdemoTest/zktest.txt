NAME
    zktest -- An unit test script for ZK

SYNOPSIS
    zktest [-bhq] [-acit] [expression]

DESCRIPTION
    zktest is a small utility fot unit test of zkdemoTest. These tests are made by
    using Selenium Test Suite (http://seleniumhq.org/).

OPTIONS
    -a, --all
        Do test(s) for all the cases in zkdemoTest.

    -b, --browser
        Open the test report in browser after doing test cases.

    -c, --category=category
        Do test(s) cases starting with category, only one category can be selected.

    -h, --help
        Show this help.

    -i, --id=id
        Do test(s) specific case which have id.

    -q, --quiet
        Do test(s) in quiet mode. This option only works when using -t(tag).

    -t, --tag=tags
        Do test(s) cases match tag(s) in config.properties of zkdemoTest.
        The tags can be multiple and non-case-sensitive, using comma as logical
        "AND" and space as logical "OR".
        i.e., zktest -t "A,Tabbox",
              zktest --tag "B Window PANEL".
