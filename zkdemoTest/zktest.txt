[1;37mNAME[m
    zktest -- An unit test script for ZK

[1;37mSYNOPSIS[m
    [1;37mzktest[m [-bhq] [-acit] [expression]

[1;37mDESCRIPTION[m
    zktest is a small utility fot unit test of zkdemoTest. These tests are made by
    using Selenium Test Suite (http://seleniumhq.org/).

[1;37mOPTIONS[m
    [1;37m-a, --all[m
        Do test(s) for all the cases in zkdemoTest.

    [1;37m-b, --browser[m
        Open the test report in browser after doing test cases.

    [1;37m-c, --category[m=[0;4mcategory[m
        Do test(s) cases starting with [0;4mcategory[m, only one category can be selected.

    [1;37m-h, --help[m
        Show this help.

    [1;37m-i, --id[m=[4mid[m
        Do test(s) specific case which have [4mid[m.

    [1;37m-q, --quiet[m
        Do test(s) in quiet mode.

    [1;37m-t, --tag[m=[4mtags[m
        Do test(s) cases match tag(s) in [4mconfig.properties[m of zkdemoTest.
        The tags can be multiple and non-case-sensitive, using comma as logical
        "AND" and space as logical "OR".
        i.e., zktest [1;37m-t[m "A,Tabbox",
              zktest [1;37m--tag[m "B Window PANEL".
